# Hypertension & Cardiovascular Risk Factors (EDA)

## Overview
This project explores relationships between **hypertension** and key cardiovascular risk factors using a real-world healthcare dataset. The analysis focuses on:
- Hypertension prevalence across **BMI categories**
- Differences in **stroke rates** between hypertension groups

This is an **exploratory data analysis (EDA)** project, not a predictive modelling project.

## Tools
- Python
- pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Key Insights
- Hypertension prevalence increases across higher BMI categories in this dataset.
- Stroke prevalence differs between individuals with and without hypertension.
- Findings are descriptive and do not imply causation.

## Dataset
Dataset sourced from Kaggle and not included in this repository.
